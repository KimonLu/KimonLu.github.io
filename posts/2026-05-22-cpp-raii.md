# C++ RAII 小结

> 这是一篇占位文章,用来演示博客卡片和阅读页,正文待补充。

RAII(Resource Acquisition Is Initialization)的核心:**资源的生命周期绑定到对象的生命周期**,构造时获取、析构时释放。

```cpp
#include <fstream>
#include <mutex>

void demo() {
    std::ofstream file("log.txt");        // 构造即打开
    std::lock_guard<std::mutex> lk(mtx);  // 构造即加锁
    // ... 中途 return / 抛异常都不会泄漏
}   // 离开作用域自动关闭文件、解锁
```
