# 个人主页编辑指南

日常维护只需要改 `index.html`(内容)和各 `js` 文件顶部的常量(参数)。本文按"你想改什么"来组织。

## 目录结构

```
KimonLu.github.io/
├─ index.html        # 页面结构与全部文案(改内容主要在这里)
├─ css/style.css     # 全部样式;文件开头是两套主题的颜色变量
├─ js/
│  ├─ main.js        # 入口,按序初始化各模块
│  ├─ i18n.js        # 中英切换
│  ├─ theme.js       # 白天/夜间主题
│  ├─ effects.js     # 打字机 / 技能条 / 背景视差
│  ├─ charcloud.js   # 字符云(光标尾迹)
│  ├─ clawd-pet.js   # Clawd 主桌宠状态机
│  └─ clawd-mini.js  # mini Clawd 彩蛋
├─ assets/
│  ├─ background.jpg # 背景图
│  └─ clawd/         # 桌宠素材(SVG + 1 个 GIF)
└─ 编辑指南.md        # 本文档
```

## 双语机制(必读)

页面所有文案都写在 HTML 元素的 `data-en` / `data-zh` 两个属性里,切换语言时由 `i18n.js` 自动填入:

```html
<p data-en="English text" data-zh="中文文案"></p>
```

**改文案 = 同时改这两个属性**,标签中间不用写字(会被覆盖)。不需要翻译的内容(如 "Python")直接写在标签里即可,不加 data 属性。

## 1. 改个人资料

都在 `index.html`:

| 想改什么 | 位置 |
|---|---|
| 大标题 LKM | `<section class="hero">` 里的 `<h1>` |
| 标题下的方向简介 | `<p class="sub">` 的 `data-en/zh` |
| 打字机那行字 | `js/effects.js` 顶部常量 `TYPED_LINE` |
| whoami 段落 | hero 区 `<p style="margin-top:12px">` 的 `data-en/zh` |
| 技能标签(小圆片) | `.tags` 里的 `<span class="tag">`,`class="tag on"` 是高亮款 |
| 浏览器标签页标题 | `<title>` 和 `<meta name="description">` |

## 2. 改技能条

`index.html` 的 `#skillBox` 里,一条技能是一个 `.skill-row`:

```html
<div class="skill-row" data-pct="80">   <!-- data-pct = 进度条百分比 -->
  <div class="label"><span>Python</span><span class="pct">80%</span></div>
  <div class="bar"><i></i></div>
</div>
```

复制整块即可新增;`data-pct`(实际宽度)和 `.pct`(显示文字)要一起改。

## 3. 改项目卡片

`index.html` 的 `<div class="grid">` 里,一张卡片模板:

```html
<div class="proj">
  <span class="pt" data-en="项目名(英)" data-zh="项目名(中)"></span>
  <span class="pd" data-en="一句话简介(英)" data-zh="一句话简介(中)"></span>
  <span class="pl"><a href="仓库或演示链接" data-en="→ repo" data-zh="→ 仓库"></a></span>
</div>
```

复制粘贴即可加卡片;最后那个虚线框 `.proj.add` 是占位提示,不需要可整块删除。

## 4. 改联系方式

`index.html` 的 `#contact` 区 `.links` 里,一行一个 `<a>`。加社交链接把 `href="#"` 换成实际地址即可。

## 5. 桌宠相关

### 改台词
`js/clawd-pet.js` 顶部 `PHRASES`,`en` 和 `zh` 两个数组按索引一一对应,增删保持两边数量一致。

### 调行为节奏
`js/clawd-pet.js` 顶部常量(单位毫秒):

| 常量 | 含义 | 默认 |
|---|---|---|
| `FLOURISH_DELAY` | 两次待机小动作的间隔范围 | 12~25 秒 |
| `FLOURISH_POOL` | 待机小动作池(素材名,可增删) | — |
| `DOZE_AFTER` / `SLEEP_AFTER` | 挂机多久打盹 / 睡着 | 45 秒 / 2 分钟 |
| `RAPID_CLICKS` | 3 秒内连点几次会头晕 | 4 |

素材名 = `assets/clawd/` 下文件名去掉 `clawd-` 前缀和扩展名,如 `working-wizard`。

### mini 彩蛋频率
`js/clawd-mini.js` 顶部 `FIRST_DELAY`(首次出场)和 `NEXT_DELAY`(之后间隔范围)。

### 调试
浏览器控制台(F12)里可立即触发各种状态,不用干等:

```js
__clawdDebug.set('working-wizard')  // 主桌宠切任意素材
__clawdDebug.sleep()                // 立即睡着
__clawdDebug.miniPeek()             // mini 探头
__clawdDebug.miniWalk()             // mini 横爬
__clawdDebug.miniSleep()            // mini 打盹
```

## 6. 字符云手感

`js/charcloud.js` 顶部常量:`CHARS`(字符集,重复越多概率越高)、`HOLD_MS`/`FADE_MS`(停留/淡出时长)、`RADIUS` 和 `SPAWN_PER_SAMPLE`(尾迹的宽度与密度)。

## 7. 主题颜色与背景

- 颜色全部在 `css/style.css` 开头:`:root{}` 是夜间,`html[data-theme="light"]{}` 是白天,每个变量有注释。
- 背景亮度:改两套主题里各自的 `--bg-brightness`(夜间默认 0.58)。
- 换背景图:替换 `assets/background.jpg`(或改 `#bg-image` 里的 url)。
- 字符云颜色:`--char-color`(写成 `r,g,b` 数字格式)。

## 8. 本地预览与发布

**预览**(桌宠 SVG 靠 fetch 加载,直接双击 index.html 用 file:// 打开时桌宠不会显示,必须走本地服务器):

```bash
cd KimonLu.github.io
python -m http.server 8000
# 浏览器访问 http://localhost:8000
```

**发布**:

```bash
git add -A
git commit -m "更新内容"
git push origin main
```

推送后等 1~2 分钟,GitHub Pages 会自动部署到 https://kimonlu.github.io 。
